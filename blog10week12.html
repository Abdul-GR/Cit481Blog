AWS CLI

Posted by Abdul Rizwan on April 17, 2020

What is AWS CLI

In this post i will be talking about AWS CLI and its installation. What is AWS CLI? AWS CLI is an AWS Command line Interface which is an open sorce tool that let user to interface with AWS services using terminal or command line shell. There are two Versions of AWS CLI Version 1.x and Version 2.x. Version 2.x is a current version and i will be walking through this version for linux and macOS.

Installation

There are two ways to install the AWS CLI. 1. graphical interface 2. Command line.
For graphical interface download it from here.
For command line Follow these steps:

1. Download the file by running this command

curl "https://awscli.amazonaws.com/AWSCLIV2.pkg" -o "AWSCLIV2.pkg"

2. Now run this command

installer -pkg AWSCLIV2.pkg \
-target CurrentUserHomeDirectory \
-applyChoiceChangesXML choices.xml

3. Lastly, you must create a symlink file in your $PATH that points to the actual aws and aws_completer programs. By running these commands.

sudo ln -s /folder/installed/aws-cli/aws /folder/in/path/aws
sudo ln -s /folder/installed/aws-cli/aws_completer /folder/in/path/aws_completer

Conformation

You can conform the installation by running tese command and you will get the version and the path in return.

which aws
aws --version
