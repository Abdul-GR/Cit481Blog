AWS EC2

Posted by Abdul Rizwan on Feb 20, 2020

AWS EC2 Instance

Amazon Elastic Compute Cloud (Amazon EC2) is a web service that provides secure, resizable compute capacity in the cloud. It is designed to make web-scale cloud computing easier for developers. Amazon EC2’s simple web service interface allows you to obtain and configure capacity with minimal friction. It provides you with complete control of your computing resources and lets you run on Amazon’s proven computing environment.

In this blog post we will be setting up Amazon EC2 instance.

To setup an EC2 instance you will need to have an account with aws.

If you have an account you can login using this link aws login

If you dont, you need to signup with aws with this link aws signup

Setting up EC2 Instance

After siningup/login search for EC2 in Find services and click on EC2. It will take you to EC2 page.

On EC2 page on your left side choose Instances from EC2 Dashboard , Choose "Launch Instance".

Configure your instance

There are any Ami that you can choose depending on your need but for this example we will be using free tier which mean you wont get charged.

Choose an Amazon Machine Image(AMI): In this step you will be choosing your Machine Image. Choose which saying "Free tier eligible" example: Amazon Linux 2.

Choose an Instance Type: In this step you will be choosing memory, CPU. Choose the one saying free tier eligible like t2.micro. Click Next: configure Instance Details.

This is just a simple EC2 installation so just click Next and give 8GB storage and go to step 6.

Configure Security Group: select SSH rule and give port 22. This is your firewall, who you want to go in and out. Click Review Launch instance.

Review Instance Launch: Here you view you instance what you have mad if something to change you can go back and change them. Click Launch.

A window will pop up, Choose creat a new key pair and give key name and download it. Save this in a safe place as we will later use this file to log in to the instance. Click Launch Instances.

Click on view my instances. it will take you to your instaces page where you will see the status of your Instance that you just created.

You need to gave permistion to your key you just downloaded by running this command in your terminal.

chmod 400 keyname.pem

You can ssh into your instance by your DNS or the IP you can find in Description and run this command in terminal.

ssh -i "/path/to/your/key/keyname.pem" ec2-user@IPorDNS

You can see this information by clicking on connect it will show you how to ssh into your machine.

Terminate Instance

EC2 is free to start but you should terminate your instances after you are done so you dont get charged extra if you choose not free tier. To do that click on Action and select terminate.

This will Terminate you selected instance.
