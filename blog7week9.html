AWS EC2 Instance using Terraform

Terraform is command line tool for building, changing, and versioning infrastructure, it supports popular cloud providers including Amazon AWS. Configuration files describe the infrastructure components needed to run a single application, for example, an EC2 instance with an attached EBS volume.

Prerequisites

1. Your system must have terraform installed. if its mac you can install it by using this command:

brew install terraform

2. AWS CLI must be installed.

If you have these two we can go head and start building EC2 instance using terraform.

EC2

EC2 stands for Elastic Compute Cloud â€” service that provides scalable computing capacity in the Amazon Web Services (AWS) cloud. Using EC2 you can launch virtual servers, setup networking and security for them, attach storage, public IP addresses or associated domain names.

Build EC2

In thtis example we will be building basic EC2 instance using terraform. To do that you need this code and save it as a .tf file.

provider "aws" {
region = "us-west-2"
}
resource "aws_instance" "web" {
ami = "ami-12345678"
instance_type = "t1.micro"
tags {
Name = "example1"
}
security_groups = [ "${aws_security_group.my_security_group.id}" ]
}

After adding this save the file and open the terminal and go to the directory where you save this .tf file.

Run this

terraform init
terraform plan
terraform apply
