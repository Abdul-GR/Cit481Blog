AWS CLI Configuration

In this blog i will be showing you how to configure AWS CLI on ur computer and its terms. It is very simple and easy.

So on your computer go to your terminal and type this command:

aws configure

It will prompt you few informations like access key, secret access key, AWS Region, and output format. This information you can find on your AWS account.

Access key and Secret Access Key you will get when you creat an IAM user or role this information is very important and must be kept in safe place because if you lose them you cannot get them back and you will need to creat a new IAM role. You can download this information as a .csv file.

AWS reagion is a server you will be picking to send your requests. You want the region closest to you but you can chose any region. You can find this in ur AWS account in upper right corner like us-west-2

The Default output format specifies how the results are formatted. The value can be any of the values in the following list. If you don't specify an output format, json is used as the default. There is json, yaml.

You can creat more than one profile to do that run this command : aws configure --profile produser and rest the steps are same as above.
