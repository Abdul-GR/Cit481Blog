Apache2 installation

Posted by Abdul Rizwan on Jan 30, 2020

This week blog will be explaining how to install apache http server on unix and unix-like system. What is apache? Apache is a free web server software that powers a big amount to internet websites. It is also known as “Apache HTTP server” and its developed by Apache Software Foundation.

To install a new software, it is good practice to check and update your software package to see if they are up to date. To update your local software package, open your terminal and type:

sudo apt-get update

once the updating is finished we are ready to install apache. To install apache, type this command in your terminal.

sudo apt-get install apache2

After the installation is done make run this command to make sure it was successfully installed

apache2 -v

Once you get the version it means you have installed apache successfully.

To run the apache, you will need to run this command in your terminal.

service apache2 start

To check the status run this command.

service apache2 status

At this point apache should be running and installed successfully. To verify it, we can go on localhost in our web browser.

Localhost:8080

And you should get a reply back with a webpage if you are using ubuntu it will say “Apache2 “Ubuntu Default page”.

Run this command in your terminal to stop apache.

service apache2 stop

To restart it, run this command.

service apache2 restart.
