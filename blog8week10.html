NeoLoad performance software

NeoLoad is a powerful load and performance testing software solution designed for web and mobile applications. It is a tool that provides testers and developers test designs and realistic performance of user behavior. It allow an devekoper to add large number of simulate user to test the behaviour of the server. It allows user to check the service he or she delveloping before launching it so user can fix major issues before it become a major issues and become expensive to fix later.

NeoLoad is written in java and works by simulating traffic to determine application performance under load, analyze response times and pinpoint the number of the simultaneous users which the Internet, mobile application can handle.

Features

1. It can record HTTPS requst, so later you can minupilate.

2. It supports ActiveX components as long as communication with the server is via the HTTP protocol.

3. It supports hybrid and native mobile applications.

4. JSON and SPDY protocol are also supported.

5. It supports video straming.

6. Graphs and charts to see how the traffic is and responses are.
